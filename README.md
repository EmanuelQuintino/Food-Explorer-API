## Installation API

```bash
# Clone Project
$ git clone https://github.com/EmanuelQuintino/Food-Explorer-API.git

# Install Dependencies
$ npm install

# Install Prisma Client
$ npm install @prisma/client

# Run prisma migrations:
$ npx prisma migrate dev

# Run prisma seed:
$ npx prisma db seed

# Run API
$ npm run dev
```
