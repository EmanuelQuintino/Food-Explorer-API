# Food Explorer API
## Technologies
- `Node.js`
- `Typescript`
- `Express`
- `JSON Web Token (JWT)`
- `Prisma`
- `Bcrypt`
- `Multer`

## Installation API

```bash
# Clone project
$ git clone https://github.com/EmanuelQuintino/Food-Explorer-API.git

# Install dependencies
$ npm install

# Install prisma client
$ npm install @prisma/client

# Run prisma migrations
$ npx prisma migrate dev

# Run prisma seeds
$ npx prisma db seed

# Run API
$ npm run dev
```
## Attention

#### Fill environment variables

```ini
DATABASE_URL=""
SECRET_TOKEN=""
EXPIRESIN_TOKEN=""
ADMIN_EMAIL=""
ADMIN_PASSWORD=""
```
